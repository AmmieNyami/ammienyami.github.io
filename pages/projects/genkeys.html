genkeys

genkeys is program that reads a configuration file containing key binding definitions and generates key binding definitions in another configuration format specified by the user. This is useful if you have multiple window managers/wayland compositors installed and want your key bindings to be the same across them.

genkeys supports generating key bindings for sway/i3, hyprland and bspwm (sxhkd).
